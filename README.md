# 🐱 Random Cat Chrome Extension

A simple Chrome Extension built with **Manifest V3** that replaces every image on a webpage with random cat images fetched from **TheCatAPI**.

---

## 📌 Features

- 🐱 Replaces all webpage images with random cat images
- ⚡ Built using Chrome Extension Manifest V3
- 🌐 Fetches random cat images from TheCatAPI
- 🖼️ Works on most websites containing images
- 🔧 Easy to install using Chrome Developer Mode

## 🚀 Installation

1. Download or clone this repository.
2. Open **Google Chrome**.
3. Navigate to **chrome://extensions/**.
4. Enable **Developer Mode**.
5. Click **Load unpacked**.
6. Select the **Random-Cat-Chrome-Extension** folder.
7. Open any webpage containing images.
8. Refresh the page and watch the images transform into random cats! 🐱

## 📖 How It Works

The extension scans every `<img>` element on a webpage and requests a random cat image from **TheCatAPI**. Each original image is then replaced with the fetched cat image automatically.
